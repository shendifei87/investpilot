# Sample Workspace

This directory contains **synthetic demo data** for InvestPilot. No real financial data, proprietary information, or actual analyst research is included. All numbers are fabricated for demonstration purposes only.

## Contents

| File | Description |
|:-----|:------------|
| `step0_quick_triage.md` | Synthetic quick triage output for a fictitious A-share company |
| `forecast_model.json` | Minimal synthetic financial forecast model |
| `step4_structured_assumptions.json` | Synthetic Step 4 assumptions matrix |
| `monte_carlo_results.json` | Synthetic Monte Carlo simulation output |
| `sample_report.html` | Small self-contained HTML research report |

## Usage

This data is used by the test suite and can also serve as a reference for understanding InvestPilot's output formats. See `docs/architecture.md` for the full pipeline description.
